# Default Workflow (Canonical)

## Source of truth
This repository (`agentic-notes`) is the **default destination** for all new writing:
- article drafts
- daily notes
- presentation notes
- references/snippets

If content starts elsewhere, move/copy it here at the end of the session.

## Folder rules
- `daily/` → quick captures and daily logs
- `articles/` → draft + revision cycle
- `presentations/` → talk notes and outlines
- `references/` → source extracts and links
- `templates/` → reusable writing templates

## Default save behavior
When creating new markdown content without an explicit destination, save in this repo first.

## Promotion flow
1. Draft in `agentic-notes`
2. Refine and finalize
3. Promote stable content to other repos/docs as needed

## Naming conventions
- Daily: `daily/YYYY-MM-DD.md`
- Articles: `articles/topic-slug.md`
- Presentations: `presentations/topic-slug.md`

## Mobile first
This repo is optimized for Obsidian + Working Copy on mobile.
