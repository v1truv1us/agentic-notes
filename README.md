# Agentic Notes (Obsidian Vault)

> **Default destination:** New notes/articles/presentation drafts should be added here by default.

Personal Obsidian-first markdown vault for:
- daily notes
- article drafts
- presentation notes
- reference captures

## Structure
- `daily/`
- `articles/`
- `presentations/`
- `references/`
- `templates/`

## Mobile workflow (iOS)
1. Clone in Working Copy
2. Open repo as Obsidian vault
3. Edit notes
4. Commit/push from Working Copy

## Sync helper
Use `scripts/sync-notes.sh` from desktop if needed.
